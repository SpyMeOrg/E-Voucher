# سكربتات إعداد نظام تتبع الأوردرات

هذا المجلد يحتوي على السكربتات اللازمة لإعداد وتشغيل نظام تتبع الأوردرات على سيرفر لينكس.

## طريقة التثبيت السريع

إذا كنت تريد تثبيت النظام بسرعة على سيرفر جديد، قم بتنفيذ الخطوات التالية:

1. قم بتسجيل الدخول إلى السيرفر كمستخدم `root` أو باستخدام `sudo`
2. قم بتنزيل سكربت التثبيت السريع:
   ```bash
   wget -O install.sh https://raw.githubusercontent.com/SpyMeOrg/E-Voucher/main/src/features/order-tracking/scripts/install.sh
   chmod +x install.sh
   ```
3. قم بتشغيل سكربت التثبيت:
   ```bash
   sudo ./install.sh
   ```
4. اتبع التعليمات التي تظهر على الشاشة

## محتويات المجلد

1. **install.sh**: سكربت التثبيت السريع الذي يقوم بتنزيل المشروع وتنفيذ سكربت الإعداد.
2. **setup.sh**: سكربت الإعداد الرئيسي الذي يقوم بتثبيت جميع المتطلبات وإعداد السيرفر.
3. **backup.sh**: سكربت النسخ الاحتياطي اليومي.
4. **voo-tracker.service**: ملف خدمة systemd لتشغيل التطبيق كخدمة.
5. **.env.example**: ملف تكوين بيئي نموذجي.

## الخطوات التفصيلية التي تتم أثناء التثبيت

1. **تثبيت المتطلبات**:
   - تثبيت Node.js 18.x
   - تثبيت PM2 لإدارة العمليات
   - تثبيت المكتبات المطلوبة

2. **إعداد الأمان**:
   - تثبيت وإعداد جدار الحماية (UFW)
   - فتح المنافذ اللازمة فقط (SSH, HTTP, HTTPS)

3. **إعداد المستخدم والخدمة**:
   - إنشاء مستخدم خاص بالتطبيق (voo-app)
   - إعداد التطبيق كخدمة نظام

4. **إعداد النسخ الاحتياطي**:
   - إعداد نظام نسخ احتياطي يومي تلقائي
   - حذف النسخ الاحتياطية القديمة (أكثر من 7 أيام)

## بعد التثبيت

بعد الانتهاء من عملية التثبيت، يجب عليك:

1. تعديل ملف البيئة لإدخال إعدادات البريد الإلكتروني والواتساب الخاصة بك:
   ```bash
   nano /home/voo-app/voo-order-tracker/.env
   ```

2. إعادة تشغيل الخدمة:
   ```bash
   systemctl restart voo-tracker
   ```

3. التحقق من حالة الخدمة:
   ```bash
   systemctl status voo-tracker
   ```

## الصيانة الدورية

- يتم إجراء نسخ احتياطي يومي في الساعة 12 منتصف الليل
- يمكن الوصول إلى النسخ الاحتياطية في المجلد: `/home/voo-app/backups`
- يتم الاحتفاظ بالنسخ الاحتياطية لمدة 7 أيام فقط
