# نظام تتبع الأوردرات - Voo Payment

## وصف المشروع
نظام متكامل لتتبع وإدارة أوردرات تحويل فودافون كاش من الإمارات إلى مصر. يقوم النظام بمراقبة البريد الإلكتروني تلقائياً واستخراج البيانات من ملفات Excel المرفقة وإرسال إشعارات فورية على الواتساب.

## المميزات
- مراقبة تلقائية للبريد الإلكتروني
- استخراج البيانات من ملفات Excel
- حساب الرسوم تلقائياً حسب شركة المحمول
- إرسال إشعارات فورية على الواتساب
- عرض الأوردرات في جدول منظم
- إمكانية فلترة حسب التاريخ

## المتطلبات التقنية
### السيرفر
- Node.js (الإصدار 18 أو أحدث)
- خادم VPS أو Dedicated Server
- نظام تشغيل Linux (مثل Ubuntu)

### المكتبات المستخدمة
- node-imap: لقراءة البريد الإلكتروني
- xlsx: لمعالجة ملفات Excel
- whatsapp-web.js: لإرسال إشعارات الواتساب

## آلية العمل
1. **مراقبة البريد الإلكتروني**
   - مراقبة البريد كل دقيقة
   - البحث عن رسائل من no-reply@voopayment.com
   - استخراج ملفات Excel المرفقة

2. **معالجة البيانات**
   - قراءة ملف Excel
   - استخراج:
     * رقم العملية
     * رقم الموبايل
     * المبلغ
     * التاريخ
   - معالجة رقم الموبايل (إضافة 0 إذا لزم الأمر)
   - تحديد شركة المحمول

3. **حساب الرسوم**
   - فودافون: 1 جنيه ثابت
   - باقي الشركات: 0.5% بحد أقصى 15 جنيه
   مثال:
   - مبلغ 1000 جنيه = 5 جنيه رسوم
   - مبلغ 4000 جنيه = 15 جنيه (الحد الأقصى)

4. **إرسال إشعارات الواتساب**
   تنسيق الرسالة:
   ```
   🔔 أوردر جديد:
   ⏰ [رقم_العملية]     [التاريخ] [الوقت]
   📱 رقم الموبايل: [رقم_الموبايل]
   💰 المبلغ المدفوع: [المبلغ]
   💰 الرسوم: [الرسوم]
   💰 المبلغ المطلوب تحويله: [المبلغ_النهائي]
   ```

## متطلبات السيرفر
1. **المواصفات الدنيا:**
   - CPU: 1 core
   - RAM: 1GB
   - مساحة تخزين: 20GB
   - نظام تشغيل: Ubuntu 20.04 LTS

2. **البرامج المطلوبة:**
   - Node.js
   - PM2 (لإدارة العمليات)
   - Git

3. **الأمان:**
   - جدار حماية (Firewall)
   - شهادة SSL
   - نظام نسخ احتياطي
